# wa-fuel-price-analytics
An end to end data analytics project analysing fuel price patterns across Western Australia and recommending the cheapest locations and times to refuel.
## Project objectives
- Analyse fuel price trends across Perth and Western Australia
- Identify the cheapest suburbs, brands, and fuel stations
- Explore weekly fuel price cycles
- Build a fuel saving calculator
- Develop an interactive dashboard

## Data source
FuelWatch Western Australia

## Technologies
- Python
- Pandas
- SQL
- Matplotlib
- Streamlit
